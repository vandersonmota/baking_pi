Supported Boards for ARM EABI

The following linker scripts are provided for arm-none-eabi.

ARM M-profile Simulator

  • Processor name: Cortex-M3
  • Processor options: -mcpu=cortex-m3 -mthumb
  • Memory regions: ram

Unhostedgeneric-m.ld__cs3_reset_generic_m
